# fastapi_heroku

Illustrates how to deploy a barebones FastAPI via Heroku.

## To run locally

    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt

Then, run via uvicorn:

    uvicorn main:app --reload

## To deploy to heroku

    heroku login
    heroku create
    git push heroku main
    heroku open
